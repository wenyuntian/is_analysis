# 接口：login  [返回](../README.md)
用例： [登录](../用例/登录用例.md)

- 功能：
    登录到首页。
    
- 权限：
    访客（学生、教师、管理员）。    
    
- API请求地址： 
    接口基本地址/v1/api/login

- 请求方式 ：
    POST

- 请求实例：

        {
            "id":"201510414121",
            "password":"daydaywen",
            "type":"1"
        }
        
- 请求参数说明:        

  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |id|学生学号或者老师的工号，由type决定。|
  |password|用户的密码，不是原文，是加密后的字符串| 
  |type|用户类型，学生、老师、或管理人员（1代表学生，2代表老师，3代表管理员）|
  
- 返回实例：

        { 
            "status": false,
            "info": "密码不正确",    
        }
 
- 返回参数说明：    
 
  |参数名称|说明|
  |:---------:|:--------------------------------------------------------|      
  |status|bool类型，true表示正确的返回，false表示有错误|
  |info|返回结果说明信息|
